### 🔥 精选项目 | Featured Projects

#### [BlinkMD](https://github.com/xxoxxsxx/BlinkMD)
本地优先的轻量级 Markdown 编辑器，启动快、内存低、键盘驱动。

**技术栈**：Tauri 2 + React 18 + TypeScript + Vite + CodeMirror 6  
**亮点**：分栏/编辑/预览模式、原生文件操作、GFM 支持、未保存变更保护、大文件优化  
![Stars](https://img.shields.io/github/stars/xxoxxsxx/BlinkMD?style=flat-square&color=58A6FF)

**亮点**：虚拟列表缩略图、键盘快捷键、单实例复用、异步加载  
![Stars](https://img.shields.io/github/stars/xxoxxsxx/lume?style=flat-square&color=58A6FF) （近期持续维护）
